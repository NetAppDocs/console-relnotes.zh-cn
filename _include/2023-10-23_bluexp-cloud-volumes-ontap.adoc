=== Azure 中支持 HA 多可用区部署的新区域

Azure 中的以下区域现在支持Cloud Volumes ONTAP 9.14.0 及更高版本的高可用性多可用区部署：

* 澳大利亚东部
* 东亚
* 法国中部
* 北欧
* 卡塔尔中央
* 瑞典中央
* 西欧
* 美国西部 2


有关支持多个可用区域的所有区域的列表，请参阅 https://bluexp.netapp.com/cloud-volumes-global-regions["Azure 下的全球区域地图"^]。
